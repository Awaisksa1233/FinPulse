iOS Shortcut Setup

Add expenses with natural language!

AI API Endpoint

{{ server_url }}/api/ai

Just say "Paid 50 for lunch" and AI will parse it!

Step-by-Step Setup

  1. Open Shortcuts app on your iPhone
  2. Tap + to create new shortcut
  3. Add action: "Ask for Input" → Text → Prompt: "What did you spend?"
  4. Add action: "Get Contents of URL"
  5. URL: {{ server_url }}/api/ai
  6. Method: POST
  7. Request Body: JSON
  8. Add field: text → Select "Provided Input"
  9. Add action: "Get Dictionary Value" → Key: "message"
  10. Add action: "Show Result"
  11. Name: "Add Expense"

Example Commands

  • Paid 50 for lunch
  • Spent 200 on groceries
  • Received 5000 salary
  • Transfer 100 from wallet to bank
