iOS Shortcut Setup

Add expenses with natural language using Siri or Shortcuts!

🔒 Secure API Configuration

📱 Step-by-Step Setup Guide

  1. Open the Shortcuts app on your iPhone.
  2. Tap the + icon in the top right to create a new shortcut.
  3. Add the action: "Ask for Input" (Type: Text, Prompt: "What did you spend?").
  4. Add the action: "Get Contents of URL".
  5. Set the URL to your Personal API URL copied above.
  6. Change the Method to POST.
  7. Set the Request Body to JSON.
  8. Add a new text field: Key: text, Value: Select "Provided Input".
  9. Add the action: "Get Dictionary Value" (Key: message).
  10. Add the action: "Show Result" (displays the AI's confirmation).
  11. Name the shortcut "Add Expense" and save!

💡 Example Voice Commands

You can trigger the shortcut by saying "Hey Siri, Add Expense", and speak phrases like:

  • Paid 50 for lunch
  • Spent 200 on groceries
  • Received 5000 salary
  • Transfer 100 from Wallet to Bank
